model_definition

True

hex

25

format: 0x[nn pp cc cc pp cc cc ...] First entry is 100% and last entry is 0%

  • nn: number of entries, up to 8 entries
  • pp: percent
  • cc cc: count value, 2 bytes in little endian

Example: 0x03, 0x64, 0xD0, 0x07, 0x37, 0x28, 0x0A, 0x00, 0xA0, 0x0F --> 3 entries, percent=100% counts=2000, percent=55% counts=2600, percent=0% counts=4000

Model definition with up to 7 intermediate linear interpolation points
